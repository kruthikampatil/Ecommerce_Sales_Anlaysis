# Ecommerce Sales Analysis Project

## Project Overview

This project analyzes ecommerce sales data using Excel, SQL, Power BI, and Python to generate meaningful business insights and visualize sales performance.

## Tools & Technologies Used

* Microsoft Excel
* MySQL
* Power BI
* Python
* Pandas

## Project Modules

### 1. Excel Project

* Data cleaning
* Data formatting
* Basic sales analysis

### 2. SQL Project

Performed SQL queries for:

* Total sales calculation
* Top sales records
* Region-wise profit analysis
* Average sales analysis
* Total orders count

### 3. Power BI Project

Created interactive dashboard for:

* Sales analysis
* Profit analysis
* Regional insights
* Category performance

### 4. Python Project

Performed:

* Data loading
* Data analysis
* Business insights generation

## Dataset

The project uses an Ecommerce Sales Dataset in CSV and Excel formats.

## Key Insights

* Identified high-performing sales regions
* Analyzed profit trends
* Found top-performing products
* Visualized sales performance using dashboards
